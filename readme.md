# Программа-помошник для анализа доходов и расходов.

Небольшой pet-проект для демонстрации базовых навыков Python, pandas и matplotlib.

---

## Цель проекта:
Программа анализирует CSV-файл с личными расходами и выводит:
1. общую сумму трат;
2. средний расход;
3. сводку по категориям;
4. визуализацию в виде графика.

---

## Используемые технологии:
1. Python 3.10+
2. pandas
3. matplotlib

---

## Структура проекта
```
budget-analyzer/

│

│   └── report.csv
│
├── budget_analyzer.py
│
└── README.md
```
---

## Запуск
1. Установить зависимости:
   ```bash
   pip install pandas matplotlib

2. Запустить анализ:
   ```bash
   python budget_analyzer.py
 
3. Результаты: 
   На экране появится статистика и график. Отчёт сохранится в output/report.csv.
